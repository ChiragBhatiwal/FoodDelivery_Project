# 🍔 Food Delivery App

A **scalable food delivery application** with **GraphQL, Kafka, Redis, and JWT authentication**, enabling real-time restaurant discovery, order tracking, and efficient data processing.

## 🚀 Features
- **GraphQL API** for efficient data fetching  
- **Geolocation-based filtering** for nearby restaurants  
- **Kafka for real-time event-driven architecture**  
- **Redis caching** for optimized performance  
- **JWT authentication** for secure user access  
- **Live order tracking** with real-time updates  
- **Android frontend** using Java & Apollo GraphQL  
- **Scalable backend** with load balancing and API rate limiting  

## 🏗️ Tech Stack
- **Frontend:** Android (Java) + Apollo GraphQL  
- **Backend:** Node.js (Express) + GraphQL  
- **Database:** MongoDB  
- **Message Queue:** Apache Kafka  
- **Cache:** Redis  
- **Authentication:** JWT
